# DiamondMetrics

A diamond export and production analytics dashboard for Botswana's major mines. Single HTML file, no setup required.

## What it does
- Tracks production and export value across Jwaneng, Orapa, Letlhakane, Karowe, and Damtshaa
- KPIs: total export value, production volume, blended price per carat, year-over-year growth
- Export value trend with a 3-month forecast (linear regression)
- Production-by-mine comparison and export destination breakdown
- Average price-per-carat trend over time
- Filterable monthly production table by mine



## Theme
White background with blue and grey accents. Playfair Display for headings, Inter for body text.

## Tech
Single-file HTML/CSS/JS + Chart.js (via CDN). Data is synthetic but modeled on realistic production and pricing patterns.
